# Word template for capability statement

Place your Word template here so the backend can generate DOCX files.

**Required file name:** `Cap Statement Template V1.docx`

The document will be processed with [docxtemplater](https://docxtemplater.com/). Use placeholders in the template (e.g. `{client_name}`, `{lead_partners}`, `{awards_list}`) to be filled with data from the configuration page.

If this file is missing, the "Generate" action on the configuration page will fail with a 500 error.
